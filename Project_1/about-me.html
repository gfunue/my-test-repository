More about me

My name is Gentang Edwin Funue but I prefer going by my middle name Edwin. Delivery of scalable software systems to end-users. Daily tasks consisted of design implementation, unit testing, code review and refactoring, and stacks (queue, LinkedList). Skilled and experienced with various technologies such as Java SE 8, Spring Tool Suite 4, Slack, Microsoft Teams, Canvas, and GIT Hub.
